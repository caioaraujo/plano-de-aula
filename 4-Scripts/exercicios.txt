1 - Crie um arquivo chamado script_1.py que escreve "Olá de novo, mundo!" e execute.

# python script_1.py

2 - Crie um arquivo chamado script_2.py contendo as variáveis inteiro dia, string mes, inteiro ano com os
valores de hoje e escreva "Hoje é dia (dia/mes/ano)". Dica: Use formatação de string.

# python script_2.py

3 - Crie um arquivo chamado script_3.py que recebe o nome e sobrenome e ao final escreva "Olá (nome) (sobrenome)!".

# python script_3.py

4 - Repita o exercício anterior em um novo arquivo chamado script_4.py, porém recebendo nome e sobrenome como argumentos
do script.

# python script_4.py Nome Sobrenome
